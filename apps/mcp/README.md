# @software-factory/mcp

Servidor MCP inicial do `software-factory 2.0` via `stdio`.

Tools expostas nesta fase:

- `software_factory.providers`
- `software_factory.models`
- `software_factory.run_dry`
- `software_factory.stage_dry`
- `software_factory.retrieval_dry`
- `software_factory.video_plan_dry`
- `software_factory.video_package`

Objetivo desta fase:

- expor o runtime do `software-factory` como tools agent-native
- permitir integração com clientes MCP sem depender da CLI direta
- preparar a base para um `apps/mcp` mais completo no 2.0
