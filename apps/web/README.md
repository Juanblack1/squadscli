# @software-factory/web

Cockpit inicial do `software-factory 2.0`.

Nesta fase, a interface e propositalmente simples e sem framework pesado.

Ela permite:

- verificar `health` do server
- ver `providers`
- ver `models`
- listar `workflows`

## Uso

1. Suba o server em outra janela.
2. Rode:

```bash
cd apps/web
npm run dev
```

3. Abra `http://127.0.0.1:4173`.
